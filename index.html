quickcheck4eclipse by oroszgy

QuickCheck Eclipse integration plug-in

Dependencies

Eclipse, ErlIDE

Install

To install the plug-in, you need a working Eclipse/ErlIDE, and QuickCheck installed. After starting Eclipse press Help->Install new software and type the update site name.
